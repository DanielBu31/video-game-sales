# Video Game Sales - Tableau Dashboard

## What This Is
An interactive Tableau dashboard analyzing video game sales data to show industry trends and patterns.

## What's in the Dashboard
- Sales trends over time
- Top publishers comparison
- Most popular game genres
- Console market share
- Regional sales breakdown

## Main Findings
- Clear growth patterns in the gaming industry
- Certain publishers lead the market
- Some genres are consistently more popular
- Platform competition has shifted over time
- Sales vary by region

## Dashboard Features
- Interactive filters for year, genre, and platform
- Hover details on all charts
- Clean, professional design
- Easy-to-understand visualizations
